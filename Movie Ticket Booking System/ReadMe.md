# Movie Ticket Booking System

A simple console-based **Movie Ticket Booking System** developed using **Java** and **Object-Oriented Programming (OOP)** concepts.

This project allows users to view movies, check available seats, and book movie tickets through a menu-driven interface. It is designed for beginners to understand how real-world booking systems work using Java.

---

# Features

* View movie details
* Display available seats
* Book movie tickets
* Seat availability management
* Booking confirmation
* Email validation
* Menu-driven console interface
* Admin and Customer roles
* Object-Oriented Design

# Classes Used

| Class          | Description                   |
| -------------- | ----------------------------- |
| User           | Base class for users          |
| Admin          | Handles admin functionalities |
| Customer       | Handles customer booking      |
| Movie          | Stores movie details          |
| Theater        | Manages theater and movies    |
| Show           | Handles show and seats        |
| Seat           | Represents individual seats   |
| Booking        | Stores booking details        |
| BookingService | Handles booking logic         |

# How to Run the Project

## Clone the Repository

```bash
git clone <repository-link>
```

## Open Project

Open the project in any Java IDE such as:

* VS Code
* IntelliJ IDEA
* Eclipse

## Compile the Program

```bash
javac Movie_Ticket_Booking_System.java
```

## Run the Program

```bash
java Movie_Ticket_Booking_System
```

# Booking Flow

1. User views movie details
2. Available seats are displayed
3. User enters details
4. User selects seat
5. Seat gets booked
6. Booking confirmation is displayed
